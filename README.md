iview modal的显示隐藏问题
iview 提供了全局关闭对话框 this.$Modal.remove()
但是这个方式只能用this.$Modal.confirm在modal里生成元素才能使用
直接在modal里放元素的话只能通过其他方式如vuex 改变v-model来控制显示隐藏
转载：https://www.jianshu.com/p/09d922e388b8   iview的一些坑
